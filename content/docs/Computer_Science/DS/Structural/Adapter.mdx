## Definition
The Adapter Pattern is a structural design pattern that allows objects with incompatible interfaces to work together. It acts as a bridge between two incompatible interfaces by wrapping one interface with another.

```mermaid
classDiagram
    direction LR
    class ModernLogger {
        <<Target>>
        +log(message)
    }
    class LegacyDatabaseLogger {
        <<Adaptee>>
        +logOldSystem(text)
    }
    class LogAdapter {
        <<Adapter>>
        +log(message)
    }
    class Application {
        <<Client>>
    }
    
    ModernLogger <|-- LogAdapter
    LogAdapter --> LegacyDatabaseLogger : "adapts"
    Application --> ModernLogger : "uses"
```
