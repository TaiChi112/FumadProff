# Computer Science

ยินดีต้อนรับสู่ส่วนของวิทยาการคอมพิวเตอร์ (Computer Science)

## หัวข้อหลัก

เลือกหัวข้อที่สนใจเพื่ออ่านเนื้อหาเพิ่มเติม:

- **[Artificial Intelligence (AI)](./AI/AI)** - ปัญญาประดิษฐ์และการเรียนรู้ของเครื่อง
- **[Data Structures & Algorithms (DSA)](./DSA/DSA)** - โครงสร้างข้อมูลและอัลกอริทึม
- **[Design Patterns (DS)](./DS)** - รูปแบบการออกแบบซอฟต์แวร์
- **[Computer Organization & Architecture (COA)](./COA/COA)** - สถาปัตยกรรมคอมพิวเตอร์
- **[Theory of Computation (TOC)](./TOC/TOC)** - ทฤษฎีการคำนวณ
- **[Computer Networks (CN)](./CN/CN)** - เครือข่ายคอมพิวเตอร์
- **[Human-Computer Interaction (HCI)](./HCI/HCI)** - การโต้ตอบระหว่างมนุษย์กับคอมพิวเตอร์
- **[Object-Oriented Programming (OOP)](./OOP/OOP)** - การเขียนโปรแกรมเชิงวัตถุ
- **[Functional Programming (FP)](./FP/FP)** - การเขียนโปรแกรมเชิงฟังก์ชัน
- **[Software Engineering (SWE)](./SWE/SWE)** - วิศวกรรมซอฟต์แวร์
- **[Digital Image Processing (DIP)](./DIP/DIP)** - การประมวลผลภาพดิจิทัล
