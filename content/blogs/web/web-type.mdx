# Understanding Web Typography
- Static / JAMStack Landing & Docs – หน้าเนื้อหานิ่ง/เอกสาร, สร้างจาก Static Site Generator
- Blog / CMS – โพสต์บทความ, แท็ก, คอมเมนต์, workflow เขียน/รีวิว
- E-commerce / Marketplace – สินค้า, ตะกร้า, จ่ายเงิน, แคมเปญ/คูปอง
- SaaS / Dashboard (Multi-tenant) – บริการออนไลน์, subscription, role/permission
- Real-time / Collaboration – แชต, whiteboard, presence, OT/CRDT
- Social / Community / Forum – โพสต์/ฟีด, notification, moderation
- Booking / Ticketing – คิว/สล็อตเวลา, ตารางทรัพยากร, payment/ยกเลิก
- Education / LMS – คอร์ส, quiz, grading, progression
- Media/Streaming – วิดีโอ/เสียง, CDN, adaptive bitrate, playlist
- Admin Panel / Backoffice – CRUD หนัก, audit log, data quality
- API-first / Headless – ให้ service, web เป็น client หลายตัว (web, mobile)
- Portal/Docs Dev – คู่มือ API, versioning, search, code samples
- PWA / Offline-first – ทำงานได้แม้ออฟไลน์, sync conflict
- Microservices Platform – แยกโดเมน, event-driven, service contract
